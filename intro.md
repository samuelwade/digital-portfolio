````{tab-set}
```{tab-item} Things I learned
- Jupyter Book 2 removes a lot of friction by removing the need to manually rebuild after editing.
- Jupyter Book is going to be very useful for building reports in a more robust and easily updatable format without relying on awkward proprietary tools like Adobe InDesign or Canva.
```

```{tab-item} Things I found difficult
- Pushing ahead with Jupyter Book 2 made a few things more complicated than they needed to be, but on balance I'm glad I chose it.
- I started off trying to work in Jupyter Lab as much as possible because it's a focus of the course, but found its file and tab navigation awkward, especially as it was nested within and sometimes clashing against the browser's own UI. Editing notebooks in Jupyter Lab but using separate editor and terminal applications (Zed and Ghostty) elsewhere worked better for me.
````
